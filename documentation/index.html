Projet de développement d’une interface de tri de résultats pour le programme FitModel

Introduction

Ce programme a pour but de permettre la visualisation des résultats obtenus grâce à FitModel (développé par M. Guindon Stéphane) et d’avoir accès à un système de requêtes pour mettre en évidence les changements de régime de sélection d’intérêt sur les arbres.

Cette interface graphique s’inscrit dans le cadre d’un projet du Master 2 Bio-informatique de l’Université de Lyon 1.

Ce programme est disponible sur Windows, Mac et LINUX.

Prérequis

A ce jour, FitModelQt ne permet pas de lancer directement FitModel depuis l’interface. Cela pourra être implémenté ultérieurement. Il est donc impératif d’avoir au préalable généré les résultats de FitModel pour les visualiser dans l’interface graphique.

La documentation pour l’installation de FitModel est disponible ici.

L’interface graphique prend en entrée un arbre au format Newick et génère les arbres directement dans son interface. A partir de ces données ainsi générées, il est possible de faire des requêtes sur ces arbres. Le lancement de l’interface graphique est disponible via un fichier exécutable (dossier "binary"). Une fenêtre s’ouvre, semblable à celle-ci :


ATTENTION ! Dans le cas où l’exécutable ne se lancerai pas correctement, il est possible de lancer l’interface grâce au logiciel Qt Creator. Les détails de l’installation de ce logiciel sont disponibles ici. Une fois le logiciel Qt Creator installé, dans l’onglet « Accueil », cliquez sur « Open Project » et ouvrez le fichier se FitModelQt.pro se trouvant dans le dossier « src » (/fitmodelqt/src/FitModelQt.pro). Vous pouvez ensuite exécuter le programme (Ctrl + R, ou cliquez sur la flèche verte en bas à gauche de l’interface de QtCreator).

Spécificité MAC OS

A l'execution du fichier exécutable sur MAC OS, il vous sera peut être demandé, si vous le souhaitez, d'ouvrir cette application (car l'interface graphique n'est pas développée par MacOS). Cliquez sur ouvrir puis rendez vous dans les "préférences systèmes", cliquez sur "sécurité et confidentialité"". En bas de l'onglet général vous pouvez autoriser l'application FitModelGui a s'ouvrir. Une fois cela fait l'application s'ouvrira.

De plus, il existe une spécifité si vous possédez la version Mojav de OSX. L'application ne fonctionnera correctement qu'avec le thème clair (onglet général dans les "préférence systèmes").

Processus

Lancer une visualisation :



Pour lancer une visualisation des arbres phylogénétiques, il faut cliquer sur « Viewer » lorsque vous vous trouvez sur la page principale. Le logiciel prend en entrée un fichier au format Newick, fournit par le logiciel FitModel.

  • Optionnel : un fichier de statistiques peut aussi lui être fourni.






Note :

Il y a autant d’arbres que de codons dans la séquence analysée. Les couleurs correspondent au régime de sélection suivant :
  • Vert (w1) : sélection neutre
  • Bleu (w2) : sélection purificatrice
  • Rouge (w3) : sélection positive


ATTENTION ! Les données au format Newick générées par l’ancienne version de FitModel ne sont pas compatibles avec les nouvelles versions de FitModel sur laquelle nous avons basé notre interface graphique.




La sélection sur les arbres :

Il vous est possible de sélectionner une partie de l'arbre ou la totalité de celui-ci (grâce à l'onglet "Whole tree"). Notez que les queries qui vous allez exécuter par la suite se feront sur votre sélection.

Lorsque vous avez sélectionné une zone avec votre curseur, la sélection apparaît alors en jaune.





La recherche sur les arbres (les queries) :



Il est possible désormais de faire une recherche de changement de régime de sélection ou de conservation de celui-ci sur les arbres. En premier lieu, il vous est possible de sélectionner seulement une région d’intérêt de l’arbre ou l’arbre tout entier, comme ci-dessous.

Il vous est tout à fait possible de rentrer par exemple la querie "from w3 to w2" ou encore "from w1 to w3", par exemple. Toutefois, cette version ne permet pas encore de combiner les queries. Une version ultérieure en donnera la possibilité. Cette liste de propositions à l'écran des "Snippets" n'est donc pas exhaustive.
